# SQL Sales Analysis Project

## Dataset
AdventureWorks2012 sample database from Microsoft.

## Project Objective
The objective of this project is to analyze sales data and customer information using SQL queries.

## Tasks Performed

1. Retrieved customer contact details using multiple JOINs.
2. Filtered sales orders from May 2011.
3. Analyzed order details using relational joins.
4. Calculated total sales for May 2011.
5. Calculated monthly sales for the year 2011.
6. Analyzed total purchases made by a specific customer.

## SQL Skills Used

- SELECT statements
- JOIN operations
- WHERE filtering
- Aggregations (SUM)
- GROUP BY
- ORDER BY

## Tools Used

- SQL Server
- AdventureWorks dataset
